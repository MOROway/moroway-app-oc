#  MOROway App OC

The MOROway App OC ("Open Cordova") variant brings the [MOROway App](https://github.com/MOROway/moroway-app) to Android.

## Download

Available at [F-Droid](https://f-droid.org/de/packages/de.moroway.oc/).
